# Fish Market SQL Analysis

## Project Overview

This project analyzes fish market data using MySQL and phpMyAdmin. The analysis focuses on fish species distribution, weight characteristics, length measurements, and size classification.

## Dataset

The dataset contains information about various fish species including:

- Species
- Weight
- Length1
- Length2
- Length3
- Height
- Width

## Tools Used

- MySQL
- XAMPP
- phpMyAdmin

## SQL Analysis

The project answers the following questions:

1. Which fish species appears most frequently?
2. Which species has the highest average weight?
3. Which species has the largest average length?
4. How are fish distributed across size categories?
5. What are the maximum and minimum weights for each species?

## Key Findings

- Perch is the most common species in the dataset.
- Sturgeon has the highest average weight.
- Fish measurements vary significantly across species.
- Most fish fall into the medium size category.
